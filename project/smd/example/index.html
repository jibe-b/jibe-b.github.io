Explanation

The Smd syntax may be used to anotate an existing text. This enables to extract RDF triples, that can be stored in a triple store and queried. But a more immediate output is the visualisation of the graph.

The introduction of this Wikipedia article has been annotated this way. You may notice that no effort has been done to use real URIs. Only a CamelCase equivalent of the annotated text has been used. Yet, the SMd syntax enables to use a regular URI and to add an alternative text, only used for display.

@wiki:OpenAccess"Open Access" is §related to @wiki:Budapest..

Open Access is related to Budapest.

Use it yourself

The production of the graphs and RDF triples is easy. Just annotate a text with SMd syntax, making sure the text starts with a title.

Then run

. smd.sh full <file> circo

then open <>.html in your browser
