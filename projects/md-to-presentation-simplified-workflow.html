Convertir un texte écrit en markdown en texte pour une présentation

Idée lors de l’écriture

Lors de l’écriture d’un texte, il est possible de produire rapidement et sans effort une présentation employant la même structure de titres et des élements présents dans le texte.

Ce qu’il faut faire

Pour cela, il suffit d’indiquer les éléments à conserver en les mettant en exergue à l’aide de la syntaxe markdown : _en écrivant comme ceci_.

Par ailleurs, au moment de la traduction dans un autre langage, il faut modifier le fichier de texte, par exemple en p_laçant tous les blocs de texte qui ne seront pas en exergue en commentaires_.

Lignes de code à écrire

La commande pandoc –filter=EXECUTABLE permet de travailler directement sur le fichier JSON produit par pandoc au cours du traitement du fichier.

Le script python en projet doit réaliser assurer cette modification du fichier JSON.

Usage

Le script python est à placer dans le PATH de l’utilisateur.

La commande pandoc -s –filter=clean_text_for_impress.py –revealjs input.md –revealjs output.html produit une présentation utilisant la librairie javascript revealjs.
