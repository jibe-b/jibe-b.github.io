Resources on Linked Data

First, a few words

Here are resources that enable one to start learning about Linked Data. These resources are mainly W3C standards, that is to say tough documents.

I strongly recommend you to read them quickly and look for easier documents, tutorials on the web. Details in these W3C documents are to be read when you know more about Linke Data.

 Clé de voute : RDF

  • quick approach : RDF introduction

Goal = Data representation => RDF in detail

  • RDF quick documentation : RDF Primer (light documentation)
  • RDF W3C standard : RDF Concepts and Abstract Syntax
  • RDF/Turtle syntax (nice syntax for humans) : RDFa/Turtle
  • RDF/XML syntax (XML syntax) : RDF/XML
  • N-ary relations (complex sentences) : W3C recommendation

Goal = publication => RDFa

  • quick approach : RDFa Primer
  • RDFa standard : RDFa Core

Goal = general data representation => ontologies

  • quick approach of OWL : OWL primer
  • standard : OWL référence

Requests on Linked Data : SPARQL

Tools

  • ontology authoring : Webprotégé

  • protégé tutorial : there exist Protégé tutorials but I don't like the most famous ones so, just look for one you like!

  • data relations visualisation : VOWL

  • visualisation of linked data : RelFinder
